<h1 align="center">ZSH Shell Config File</h1>

<h2 align="center">Installation</h2>

```bash
git clone https://github.com/TheWisker/ZshConfig
cd ZshConfig
bash setup.sh
cd ..
rm -fr ZshConfig
```


<h2 align="center">Features</h2>

* Some aliases
* Prompt themes
* History search
* Autocompletion
* Ls command colors
* Zoxide improved cd
* Syntax highlighting
* Home, end & supr fixed
* Tab cycling completion
* Autorehasing on pacman update
